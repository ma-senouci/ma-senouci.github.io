# ma-senouci.github.io

Personal portfolio website focused on Python, LLM, RAG, and multi-agent systems.

Live site: https://ma-senouci.github.io

## About

This repository contains my personal portfolio website built with a simple static stack and published with GitHub Pages.

The site presents:
- my recent AI and LLM projects
- my technical background
- a short publications overview
- links to GitHub, LinkedIn, Google Scholar, demos, portfolio (RAG), and CV
